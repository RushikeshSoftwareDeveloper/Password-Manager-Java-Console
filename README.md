# 🔐 Password Manager (Java Console Application)

## 📌 Project Description
A simple console-based Password Manager developed using Core Java.  
The application allows users to store and retrieve passwords securely using file handling and basic encryption techniques.

## 🚀 Features
- Store website credentials
- Retrieve saved passwords
- Basic password encryption
- Secure file storage
- Console-based interface

## 🛠 Concepts Used
✔ File Handling (FileWriter, BufferedReader)  
✔ Object-Oriented Programming (Classes & Methods)  
✔ Basic Encryption Logic  
✔ Exception Handling  
✔ Scanner for User Input  

## 🔐 Security Logic
Passwords are encrypted before saving them to the file and decrypted when retrieving them.

## ▶ How to Run
1. Clone the repository
2. Open the project in Eclipse / IntelliJ
3. Run the main class
